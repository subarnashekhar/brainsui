Steps to Install & Cracking
---------------------------

1) Install the application and Launch with trial version.

2) Copy the jar file "jetbrains-agent.jar" to the below folder
Ex: C:\Program Files\JetBrains\WebStorm 2019.2\bin

3) Open WebStorm & Click on Help => Edit Custom VM Options
   and add the following line
   
   -javaagent:C:\Program Files\JetBrains\WebStorm 2019.2\bin\jetbrains-agent.jar
   
4) Close the WebStorm and Re-Open (must)

5) Copy the licence Code from file 'ACTIVATION_CODE.txt' and paste in webstorm in the following path.

  Help => Register => Activate Licence Code => Click on OK

6) Enjoy!!
